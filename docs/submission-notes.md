## Narrative Detection Tool

Link target: repository section for `narrative-radar/`.

Summary:
Solana Narrative Radar is a dependency-free Node.js agent that pulls GitHub repository
activity and live Solana RPC health data, scores emerging narratives, and emits a
ranked report with product ideas tied to each narrative.

## Open Innovation Track

Link target: repository section for `memomesh/`.

Summary:
MemoMesh is an agent-built Solana provenance tool. It creates canonical receipts for
agent outputs, produces a compact payload for the Solana Memo program, and verifies
anchored payloads through Solana RPC.

Demo answer:
The repository includes a generated example receipt in
`memomesh/examples/polish-article-receipt.json`; the README contains local run and
verification commands. A browser verifier is included in `memomesh/web/index.html`.
